# EcoPlate - SDG 12 Solution

An AI-powered prototype to reduce household food waste by suggesting recipes based on available ingredients.

## SDG Alignment
- **Primary:** SDG 12 (Responsible Consumption and Production)
- **Secondary:** SDG 13 (Climate Action)

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`

## Features
- Ingredient Input
- AI Recipe Suggestion
- Carbon Footprint Estimation
